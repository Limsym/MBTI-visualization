# MBTI 动态人格 · 有界轨迹模型

把你历次的 16Personalities / MBTI 测试结果,当成一条**会随时间漂移的轨迹**来重建,而不是简单平均。
单文件静态网页,无需后端、无需构建 —— 浏览器直接打开即可。

> 在线预览:把仓库部署到 GitHub Pages 后访问 `https://<用户名>.github.io/<仓库名>/`

---

## ✨ 它做了什么

输入你每次测试的五维百分比(I/E、S/N、T/F、J/P、A/T),网页会自动产出:

- **① 当前的你** —— 16Personalities 风格的性征条,圆点是当前估计,**淡色区块是 95% 可信区间**(跨过中线越多 = 这一维越"未定")。
- **② 潜在轨迹 θ(t)** —— 有界、不对称的可信带 + 每次原始测试点;可切换维度与回归时间尺度 τ。
- **③ 字母概率轨迹** —— 把"你是 T 还是 F"换成概率曲线 P(偏向某一极)。
- **④ 五维原始观测** —— 按真实日期排列,一眼看出测试节奏的疏密。
- **⑤ 类型时间线** —— 历次"变身"的类型散点。
- **⑥ 原始数据表** + **⑦ 自动解读**。

## 🚀 如何复刻成你自己的

**你通常只需要改一个文件:[`data.js`](./data.js)。** `index.html` 不用动。

### 1) 填进你的数据

`data.js` 里的 `records`,每条记录只需 **日期 + 五个百分比**(`tag` 可选,用于标注同一天的多次测试):

```js
const records = [
  {date:'2024-06-29', tag:'#1', ie:61, sn:86, tf:56, jp:60, at:63},
  {date:'2024-11-27',          ie:88, sn:84, tf:89, jp:66, at:51},
  // …
];
```

- 数值 = 偏向**正极**的百分比,正极约定为 **I / N / T / P / A**。
  例如 16Personalities 显示 "61% 外向(E)" → 内向那一极是 `100 − 61 = 39`,即 `ie:39`。
- **类型、16 型称号(逻辑学家/辩论家…)、`61% I` 这类文案、配色、日期范围、主导类型** 全部自动推导,你不用填。

### 2)(可选)写你自己的解读文案

`data.js` 里的 `CONFIG`,**每一项都可留 `null`** —— 留空时网页会**根据你的数据自动生成**一句中性但正确的话。想要自己的口吻,就把对应项写成字符串(支持 HTML):

```js
const CONFIG = {
  title:    null,          // 大标题
  subtitle: null,          // 副标题,null = "N 次测试 · 起 → 止 · 主导内核 XXXX-X"
  defaultDim: null,        // 轨迹图默认高亮维度,null = 自动选最不笃定的轴
  dimNotes: { ie:null, sn:null, tf:null, jp:null, at:null },   // 五维 tip 个人尾注
  notes:    { bars:null, prob:null, trend:null, timeline:null },// 各图下方点评
  insights: null,          // ⑦ 解读要点,或填 [{icon:'🧱', html:'…'}, …]
  verdict:  null,          // ① 底部读数文案
};
```

### 3) 查看

直接用浏览器打开 `index.html`,或推到 GitHub Pages。
(需要联网:ECharts 与字体从 CDN 加载。)

## 🧱 设计原则:为什么改 `data.js` 就够了

仓库刻意把内容分成两层:

| 层 | 是什么 | 谁来负责 |
|---|---|---|
| **A 事实** | 类型、称号、置信度、"最活跃的轴"、"某轴在上行"、密集簇、最长空档… | **引擎自动算**(`computeFacts`),随数据/τ 实时更新 |
| **B 声音** | 用什么口吻讲、强调哪条、给什么建议 | **`CONFIG` 可选覆盖**,留空则用 A 自动兜底 |

关键约束:**`CONFIG` 只承载内容,不承载结构**(它不决定页面有几节、什么顺序、用什么控件)。
所以无论加多少文案,都不会让布局变僵 —— 想重构页面只改 `index.html`,`data.js` 不挡路。

## 🔬 模型(`index.html` 内,无需改动)

每个维度独立建模,在 **logit 空间**做有界状态空间估计:

- **logit 连接** —— 可信带永远落在 0–100% 且不对称。
- **OU 均值回归** —— 没测试的时间里,把你拉回个人基线,而非外推趋势;时间尺度 τ 来自人格稳定性文献(retest r≈0.7/年 ⇒ τ≈10³ 天)。
- **Beta 似然** —— 噪声随取值变化,各维度按 NERIS 量表信度 α 不同(T/F 最吵)。
- **Kalman 前向 + RTS 后向平滑** —— 对稀疏、不规则时点求最优后验。

> 局限:这是对**汇总百分比**的建模,不是题项级 IRT;`n` 小的时候,空档处主要反映先验。单次预测命中不等于模型被证明 —— 持续命中才算。
